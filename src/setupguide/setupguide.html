Local AI Setup (Ollama)

ü¶ô
Enable Local Models
Browsers block requests to local servers like Ollama by default due to security (CORS). We provide a script to safely configure Ollama to accept requests from our extension.

‚ö†Ô∏è
CRITICAL: Before running the script, open Task Manager, find ollama.exe in the Details tab, and End Task.
The script will fail if Ollama is currently running in the background.
  1. Download the setup script from our GitHub Repository.
  2. Run setup-ollama.bat (Windows). This adds the necessary OLLAMA_ORIGINS variable.
  3. Restart Ollama.
  4. In the extension, enable the Ollama toggle and set your model.

To revert changes later, you can run the disable-ollama.bat script from the same repository.

Custom Modes & Prompts

Want the AI to act like a specific persona?

2. Click "Edit" next to any mode or create a new one.

Select any text on a webpage, right-click, and choose "Ask AI about selection". Perfect for quick definitions or translations without snipping.
